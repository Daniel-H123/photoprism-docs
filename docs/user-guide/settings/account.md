# Account Settings #

!!! tldr ""
    Since they are not needed, these settings are not available when running in [public mode](../../getting-started/config-options.md#authentication).

## Change Password ##

1. Click *Settings*
2. Click *Account*
3. Enter your current password
4. Enter your new password twice
5. Click *Change*
